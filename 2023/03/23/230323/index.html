Leetcode19: 删除链表的倒数第 N 个结点

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。
具体思路：

链表: 1 -> 2 -> 3 -> 4
n=3: 2 1 0 -1
n=4: 3 2 1 0
n=5: 4 3 2 1

我们从头到尾遍历，并且每一次当前指针时，k-1
  如果当k>0时，不存在该位置的节点
  如果当k=0时，则需要删除的节点就是头节点，返回head->Next
  如果当k<0时，则重新从头遍历，每移动一次，k+1直到n为0
需要删除倒数第k个节点的前一个节点一定是第N-K个节点，在第一次遍历的时候，K值变成了K-N，第二次遍历的时候，每移动一次K增加1，直到K为0。所以第二次遍历时，指针会停在第N—K个节点

快慢指针

寻找到倒数第K个节点，利用双指针，让快指针比慢指针超前k个节点。当快指针到达链表末尾时，慢指针恰好在倒数第K个节点。为了能够更好的删除该节点，我们应该将慢指针指向删除倒数第K节点的前驱节点。这样，当快指针指向末尾指针的时候，慢指针指向倒数第K+1个指针。
